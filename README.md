## 🧩 Tech Stack

We’re building this project using modern, scalable tools:

- ⚛️ React (with Vite)
- 🎨 TailwindCSS
- 🧭 React Router
- ⚙️ API Integration (Form & Events)
- 💬 Framer Motion
- 📁 Organized with feature-based folder structure

```bash

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
